折现值是个很重要的概念

折现值（present discounted value）指的是未来所要得到的款项，如果现在就回收，会值多少钱

  • 企业必须思考目前建立新厂房与设备的投资费用，把它和未来用这个设备生产东西可获得的报酬作比较
  • 如果你有房屋贷款，你所有的分期付款（包含利息）的总价值，恰好等于房子的折现值，也就是购买价
  • 政府制定政策时，也需要运用折现值这个概念，来分析成本和效益

企业筹资的管道

  • 留存收益（retained earnings），即利润，企业决定持有这笔钱，将它再投资于企业，而非以股利付给股东
  • 向银行借钱
  • 发行债券，债券有面额、利率和期限三项要素
  • 通过有价证券，即股票，吧公司的一部分所有权卖给股票持有人

人一生积累财富的关键是什么

人一生积累财富的关键，在于复利（compound interest）的力量

考虑一项金融投资时，要考虑报酬率、风险、流动性和税负

  • 投资风险的定义是，报酬率相对于该投资的平均期望值有多高或多低，资本市场就是用这种方式来说明“实际情况因人而异”的
  • 投资特定公司或特定债券的风险，可以借由“多样化”分散投资而减少。多样化（diversification）的意思是购买许多不同投资标的，以降低整体风险
  • 流动性是把投资转换为现金的意思
  • 税负是指税法是否有利于该投资，某些投资可免除所得税，而有些投资所得是要课税的

个人投资工具

  • 银行账户
  • 货币基金
  • 定期存单
  • 多样化的公司债投资组合或投资这类债券的共同基金
  • 多样化的蓝筹图投资组合。蓝筹股（blue chip stocks）指的是大型、知名公司的股票
  • 多样化的高成长股票投资组合
  • 房子
  • 贵金属，如黄金、白金

垄断者的本质是对勤劳者课税

四种类型竞争

  • 完全竞争（perfect competition），有很多小企业在制造几乎一样的产品
  • 垄断竞争（monopolistic competition），是指很多企业争相销售稍微不同的产品
  • 寡头垄断（oligopoly），有点接近垄断，只是并非由一家企业囊括全部的市场营收，而是少数企业在特定市场拥有大部分或全部营收
  • 垄断（monopoly），单一生产者在特定市场几乎拥有全部的营收

完全竞争

  • 价格接受（price taking），即完全竞争的企业必须接受市场给定的价格，在这种竞争环境中，价格会密切反映生产成本，因为竞争迫使价格降到最低，仅高于生产成本

垄断竞争

  • 较接近完全竞争而非独占。当很多厂商借由销售“差异化”的产品（意即产品相似，但不是一模一样）来竞争时，就会出现垄断竞争

寡头垄断

  • 较接近独占而非完全竞争，是指一些厂商在一个特定市场拥有大部分或全部的营业额

垄断

  • 单一卖家在特定市场拥有全部或大部分的营收
  • 达到垄断状态，常用用些障碍，如专利、法律；另外一个进入障碍是所谓的“自然垄断”。当这个产业出现规模经济现象，提供大型、稳定的企业胜过新进企业的优势时，就会发生自然垄断
  • 独占企业在市场有设定价格的能力，可让价格高于生产成本，因此可赚取较高的利润

是大池塘里的小鱼，还是小池塘里的大鱼

衡量竞争程度最简单的方式是四大企业集中度（four-firm concentration ratio），做法是将该产业前四大企业的市场占有率加起来

  • 四大企业集中度较高，表示竞争是有限的
  • 不够精细

赫芬达尔-赫希指数（Herfindahl-Hirschman Index，HHI，或称赫氏指数）是一个更精确的竞争程度的衡量指标

  • 考虑每家企业的市场占有率，先算出每家企业的营收占市场总营收的百分比，然后把每家企业市场占有率的平方和加起来。
  • 指数低表示市场竞争度高，反之亦然

很难界定个别市场范围

最佳的管制法或许就是解除管制

被管制的产业都有一个共同特征：必须依赖某种网络建设。兴建整体网络的成本是较高的，而经营的成本通常是较低的。如果放任这些大企业不管，结果往往会变成垄断

管制

  • 历史上，公用事业定价最常见的方法是成本加成管制法（cost-plus regulation）：精算过公司的生产成本后，允许一个较低的获利水平（通常以一般企业在竞争市场可赚到的报酬为依据），且锁定价格以便能够获取该水平的利润
  • 价格上限管制法（price-cap regulation），管制者（也就是政府）设定一个价格，让被管制的厂商在未来几年可以依次收费。当价格上限到期时，管制者会根据新的成本重新设定费率，以适用于新一轮期限
  • 在某些情况下，最佳的管制法就是解除管制（deregulation），任何的管制法都会面临经济学家所说的“管制俘虏（regulatory capture）”的危险。管制者似乎经常会发展出一种斯德哥尔摩症候群（Stockholm Syndrome），即同情受管制的厂商，以致其判断力收到蒙蔽，无法保护消费者

市场竞争的力量可以鼓励创新与提高效率，并惠及消费者。但是，在某些泾渭分明的情况下，当市场竞争无法良好运作时，政府可以扮演有用的角色，作为经济竞争的仲裁者

  • 政府也是安全标准、财务报告正确性与信息揭露的合理仲裁者。
  • 当市场力量的结果似乎不受欢迎时，真正的挑战是发掘根本问题，并据此设计对策。
  • 判断该问题是有关垄断、卡特尔、限制性的商业成规、自然垄断、再也不需要管制的产业，还是需要某种服务的低收入民众。
  • 与其采取极端赞成或反对管制的封闭态度，更明智的做法应是见招拆招。
  • 如果政府单单施以管制手段，市场通常会运作得很差；当管制手段也能尊重激励因素与市场力量时，它反而可能运作得很好

主张绝对的零污染是不可行的

外部性（externality）指在直接的买家和卖家之外，有第三方直接受到这笔交易的影响

  • 自由市场的概念，在某种程度上是基于买家和卖家会根据自身的最佳利益而行动。但当市场交易对第三方（没有选择涉入该交易）不利时，主张自由市场会使所有人都受益的论点就不成立。
  • 外部性可以是正面的或负面的
  • 污染是负外部性（negative externality）最重要的例子

污染管制政策

  • 命令与控制（command and control），规定了可合法排放污染的最大量。
    • 管制者可能会开始考虑产业利益，这是任何管制制度都可能出现的问题，例如先前讨论的管制俘虏的情境
    • 命令与控制的管制标准通常是无弹性的，甚至明确规定必须使用寿命技术来减少何种污染，也不会奖励一开始就避免污染或把污染减少至法定标准以下的创新做法
  • 对生产者每单位的污染课征污染税或污染费
  • “可交易的许可”（marketable permit）制度。可交易的许可给污染者排放某个污染额度的法律权利，且通常获许可的污染量会随着时间递减。如果污染者排放的污染量未超过许可额度，那么剩下的额度可以卖给别人。
  • 可以使用财产权（property right）做激励

当今最大的环境议题之一，是二氧化碳与其他气体排放造成的全球变暖的威胁

  • 想出减少碳排放的方法不难，真正的困难是：用市场导向、弹性的方法来执行，用最低的经济成本来限制碳排放

主张绝对的零污染是不可行的，也不够理性。合理的政策是平衡生产效益与污染成本，换言之，让生产的社会成本与社会效益彼此平衡
