在 Mac 上进行 PHP 开发，首先需要搭建开发环境。具体的，主要需要安装 Nginx 、 PHP-FPM 和 MySQL ，但由于网上各安装教程中软件版本不同和配置的差异，没有一个可以完全参照教程。遂，吸取前人搭建环境过程中的经验，总结下自己搭建环境的过程，mark。

搭建过程主要分一下几部分说明：首先是安装前系统的版本，以及将要安装的版本，接着是说明安装过程，分别包括Nginx 、 PHP-FPM 和 MySQL的安装过程，最后以一个简单的例子结束。

安装版本

MAC系统版本和安装的软件版本：

  • MAC 系统：OS X EL Captain, Version: 10.11
  • Nginx: 1.8.0
  • PHP-FPM: PHP 5.6.14 (fpm-fcgi)
  • MySQL: 5.6.26
